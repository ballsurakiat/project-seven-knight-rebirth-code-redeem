# 7+1 อัศวิน: เกิดใหม่กี่โมง? (Seven Knight Re:Birth Code Redeemer)

ระบบช่วยกรอกโค้ดคูปองเกม Seven Knight Re:Birth แบบหลายโค้ดพร้อมกัน ไม่ต้องนั่งกดเองให้เมื่อยมือ!

## ✨ คุณสมบัติ (Features)
- **🚀 เติมโค้ดแบบต่อเนื่อง**: ใส่โค้ดกี่รหัสก็ได้ ระบบจะไล่เติมให้ทีละโค้ดโดยอัตโนมัติ (ปลอดภัยด้วยระบบ Delay หน่วงเวลา)
- **💾 ระบบจดจำ UID**: บันทึก UID ไว้ในเบราว์เซอร์ได้หลายไอดี สลับไอดีเติมได้ทันที ไม่ต้องพิมพ์ใหม่
- **🛡️ CORS Optimized**: ตั้งค่า Proxy ผ่าน Vercel Rewrites ทำให้ใช้งานบนเว็บได้ลื่นไหล ไม่ติดปัญหา CORS
- **📱 Responsive Design**: ใช้งานได้ดีทั้งบนคอมพิวเตอร์และมือถือ
- **🎨 Modern UI**: ใช้ DaisyUI + Tailwind CSS สไตล์ Seven Knight Re:Birth

## 📖 วิธีใช้งาน (How to use)
1. **จัดการ UID**: ใส่ UID ของคุณแล้วกด "เพิ่ม" หรือเลือก UID ที่เคยบันทึกไว้
2. **ใส่รหัสคูปอง**: 
   - เปิด "กรอกโค้ดเอง" เพื่อวางรหัส (1 บรรทัดต่อ 1 รหัส)
   - หรือใช้โค้ดเริ่มต้นที่ระบบเตรียมไว้ให้
3. **กดปุ่ม "เติมรหัสทั้งหมด"**: ระบบจะเปิดหน้าต่างสรุปผลและเริ่มทำงานทันที

## 🛠️ สำหรับนักพัฒนา (For Developers)
โปรเจกต์นี้พัฒนาด้วย **Vite + Vue 3 (Composition API) + TypeScript**

### การติดตั้ง (Installation)
```bash
pnpm install
```

### การรัน (Development)
```bash
pnpm dev
```

### การสร้าง (Build)
```bash
pnpm build
```

## ⚠️ หมายเหตุ (Disclaimer)
- **ความเป็นส่วนตัว**: ข้อมูล UID ทั้งหมดถูกบันทึกไว้ใน `LocalStorage` ของเบราว์เซอร์ผู้ใช้เท่านั้น ไม่มีการส่งไปเก็บที่เซิร์ฟเวอร์ใดๆ
