# เซเว่นไนท์: เกิดใหม่กี่โมง? (Seven Knight Re:Birth Code Redeemer)

ระบบช่วยกรอกโค้ดคูปองเกม Seven Knight Re:Birth แบบหลายโค้ดพร้อมกัน ไม่ต้องนั่งกดเองให้เมื่อยมือ!

## คุณสมบัติ (Features)
- **ระบบจดจำ UID**: บันทึก UID ไว้ใช้ในครั้งต่อไปได้หลายไอดี ไม่ต้องพิมพ์ใหม่ทุกรอบ
- **เติมโค้ดแบบต่อเนื่อง**: ใส่โค้ดรวดเดียว ระบบจะทำการเติมให้ทีละโค้ดโดยอัตโนมัติ (หน่วงเวลา 1 วินาทีต่อโค้ดเพื่อความปลอดภัย)
- **โค้ดล่าสุด**: มีระบบเลือกใช้โค้ดล่าสุดที่หามาให้แล้ว (สามารถอัปเดตเองได้ในไฟล์ `src/data/defaultCodes.ts`)
- **ภาษาไทย**: อินเตอร์เฟสภาษาไทย เข้าใจง่าย สไตล์เป็นกันเอง

## วิธีใช้งาน (How to use)
1. **จัดการ UID**: ใส่ UID ของคุณแล้วกด "เพิ่ม" หรือเลือก UID ที่เคยบันทึกไว้จากรายการ
2. **ใส่รหัสคูปอง**:
   - เลือก "ใช้โค้ดล่าสุดเท่าที่หาได้" เพื่อใช้โค้ดมาตรฐานที่ระบบเตรียมไว้ให้
   - หรือปิดสวิตช์เพื่อพิมพ์/วางรหัสคูปองเอง (1 รหัสต่อ 1 บรรทัด)
3. **กดปุ่ม "เติมให้หมดนี่เลย!"**: ระบบจะเริ่มทำการเติมโค้ดให้ทีละรายการ
4. **ตรวจสอบผล**: ดูผลลัพธ์การเติมได้ที่ตารางด้านล่าง ว่าโค้ดไหนสำเร็จหรือมีข้อผิดพลาดอะไร

## สำหรับนักพัฒนา (For Developers)
โปรเจกต์นี้พัฒนาด้วย **Vite + Vue 3 + DaisyUI (Tailwind CSS)**

### การติดตั้ง (Installation)
```bash
npm install
```

### การรัน (Development)
```bash
npm run dev
```

### การอัปเดตโค้ดเริ่มต้น (Updating Default Codes)
แก้ไขไฟล์ `src/data/defaultCodes.ts`:
```typescript
export const defaultCodes: string[] = [
  "CODE_1",
  "CODE_2",
  // เพิ่มโค้ดใหม่ที่นี่
];
```
